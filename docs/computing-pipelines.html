Model fitting

Fit model:

penguins_fit <- linear_reg() %>%
  set_engine("lm") %>%
  fit(body_mass_g ~ flipper_length_mm, data = penguins)

Tidy model output:

tidy(penguins_fit)
# A tibble: 2 x 5
